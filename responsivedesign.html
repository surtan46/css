Responsive design

CSS media queries can be used to create responsive websites/web applications since it allows us to write css declarations for different types of media such as screen,print and speech(screen readers)

with screen size responsive design,we can take two approaches

  • Mobile-first approach

    We write styles for small devices first then do media queries for large screens. Min-width values

  • Desktop-first approach

    We write styles for large devices first then do the media queries for small
